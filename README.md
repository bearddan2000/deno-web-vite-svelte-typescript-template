# deno-web-vite-svelte-typescript-template

## Description
A svelte template for the vite packager.

## Tech stack
- deno
- svelte
- typescript
- vite

## Docker stack
- denoland/deno:alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
